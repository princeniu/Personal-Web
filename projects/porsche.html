Porsche Digital Interface

UI&UX Design Internship Summer 2023

Project Overview

During my internship at Porsche, I was tasked with redesigning the in-car digital interface for next-generation vehicles. The project focused on enhancing user experience while maintaining driver safety.

Challenge

The main challenge was to create an intuitive interface that would minimize driver distraction while providing essential information and controls.

Process

  • Conducted user research with current Porsche owners
  • Created user personas and journey maps
  • Developed wireframes and prototypes
  • Conducted usability testing
  • Refined designs based on feedback

Tools Used

Figma Prototyping User Research

Key Features

  • Customizable dashboard display
  • Voice-controlled navigation
  • Adaptive interface based on driving mode
  • Integrated safety alerts

Results

The new interface design received positive feedback from both the Porsche design team and test users, with a 40% improvement in task completion time and a 30% reduction in reported driver distraction.
